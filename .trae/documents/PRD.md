## 1. 产品概述

本系统是热处理车间渗碳淬火业务管理后台，专为热处理工厂设计的生产管理系统，用于全面管理工艺、炉次和检测全流程。系统覆盖从工艺卡片制定、装炉排产、渗碳淬火过程记录、回火处理、金相检测、硬度检验到变形矫正的完整业务链条，实现生产过程数字化、可追溯。

目标价值：实现热处理生产数据集中管理，提升工艺执行规范性，建立炉次质量全流程追溯能力。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 系统创建 | 用户管理、系统配置 |
| 工艺工程师 | 系统创建 | 工艺卡片制定与维护 |
| 生产操作员 | 系统创建 | 装炉排产、过程记录录入 |
| 质量检验员 | 系统创建 | 金相检测、硬度检验记录录入 |
| 生产主管 | 系统创建 | 全部模块查看、数据统计与追溯 |

### 2.2 功能模块

1. **仪表板**：生产概览、炉次状态统计、质量指标
2. **工艺卡片管理**：热处理工艺卡CRUD、工艺参数模板
3. **装炉排产**：零件装炉排布、炉次计划、装炉清单
4. **渗碳淬火记录**：渗碳层深度记录、淬火介质温度、油槽冷却记录
5. **回火处理记录**：回火炉温曲线、回火参数记录
6. **金相检测**：金相组织判定、检测报告
7. **硬度检验**：洛氏硬度检验、心部硬度抽检
8. **变形矫正**：零件变形量测量、校直矫正记录
9. **质量追溯**：炉次质量追溯查询

## 3. 核心流程

### 主业务流程：

1. 工艺工程师制定热处理工艺卡
2. 生产操作员根据工艺卡进行装炉排产
3. 执行渗碳淬火工序，记录过程参数
4. 执行回火处理工序，记录炉温曲线
5. 质量检验员进行金相检测
6. 质量检验员进行硬度检验
7. 超差零件进入变形矫正工序
8. 完成后生成炉次质量档案，支持追溯查询

```mermaid

    A["工艺卡片制定" --> B["装炉排产"]
    B --> C["渗碳淬火记录"]
    C --> D["回火处理记录"]
    D --> E["金相检测"]
    E --> F{"是否合格"}
    F -->|是| G["硬度检验"]
    F -->|否| H["返工/报废"]
    G --> I{"硬度合格?"}
    I -->|是| J["变形检测"]
    I -->|否| H
    J --> K{"变形超差?"}
    K -->|是| L["变形矫正"]
    L --> J
    K -->|否| M["完成入库"]
    M --> N["炉次质量追溯"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：工业深蓝色系 (#1e3a5f 深蓝工业主色，配合 #2563eb 功能强调色

- **辅助色**：#0ea5e9 浅蓝色用于数据可视化、#64748b 中性灰
- **状态色**：#22c55e 绿色(合格)、#ef4444 红色(不合格)、#f59e0b 橙色(进行中)
- **按钮风格**：直角微圆角 (radius-md，阴影层次分明，hover
- **字体**：使用思源黑体/微软雅黑，现代工业感字体
- **布局风格**：左侧导航栏 + 顶部信息栏 + 主内容卡片式布局
- **图标风格**：线性图标 (Lucide Icons)，工业简约风

### 4.2 页面设计总览

| 页面名称 | 模块名称 | UI元素 |
|-----------|----------|--------|
| 仪表板 | 数据概览 | 统计卡片网格、图表区、列表表格、渐变背景 |
| 所有模块页面 | 列表页 | 搜索过滤栏、操作按钮区、数据表格、分页 |
| 所有模块页面 | 表单页 | 分组表单、日期选择器、数值输入、提交操作 |
| 渗碳/回火 | 温度曲线 | SVG折线图、多点数据点、时间轴 |
| 装炉排产 | 装炉布局 | 网格布局、炉位示意图、拖拽交互 |

### 4.3 响应式设计

桌面端优先设计，支持 1440px 及以上最佳，1280px 自适应。表格横向滚动支持。

### 4.4 数据可视化

- 使用 Recharts 图表库
- 炉温曲线：折线图多点折线
- 统计：环形图、柱状图
